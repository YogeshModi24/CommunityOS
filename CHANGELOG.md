# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-06-29

### Changed

- **UX & Accessibility**: Replaced text-based error states in the Citizen Profile with standardized `OSStateView` boundaries. Added strict ARIA roles (`role="tablist"`) to profile navigation tabs for improved screen reader support.
- **Documentation**: Generated comprehensive portfolio assets including technical case studies, demo scripts, and architecture overviews.
- **Audit**: Completed full Heuristic Expert Audits covering UX, Accessibility, Performance, Security, and Reliability to ensure production stability.

## [1.0.0] - 2026-06-28

### Added

- Core Reporting System allowing citizens to submit infrastructure issues.
- Municipality Mission Control dashboard with real-time telemetry.
- AI Image Classification pipeline via OpenAI Vision.
- Interactive Map clustered visualization powered by Mapbox GL JS.
- Gamified Leaderboard to encourage civic engagement.
- High-performance monorepo architecture using Turborepo.
- Comprehensive REST API built with Express.js and TypeScript.
