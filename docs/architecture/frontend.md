# Frontend Architecture

The CommunityOS frontend is built using **Next.js 14** utilizing the App Router.

## Core Technologies
- **Framework:** Next.js 14 (React)
- **Styling:** Tailwind CSS + custom UI components
- **State Management:** React Context + SWR for data fetching
- **Maps:** Mapbox GL JS for WebGL rendering

## Application Structure
- `app/(dashboard)`: The Municipality Mission Control.
- `app/(marketing)`: The Citizen landing page.
- `app/feed`: The real-time live feed of issues.
- `components/`: Reusable UI components.
