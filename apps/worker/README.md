# CommunityOS Worker Workspace

This is the scaffolded background queue worker workspace for CommunityOS. It acts as the execution layer for asynchronous tasks like OpenAI visual analysis, routing alerts, and batch audits.

## Project Structure

Uses BullMQ and ioredis to execute jobs offloaded from the main API gateway, to be fully integrated in future sprints.
